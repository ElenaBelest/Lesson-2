# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом, необходими обромить его звездочкой (*)
 Например : *вот так*

Чтобы выделить текст полужирным, необходими обромить его двумя звездочками (**) 
Например: **вот так**
